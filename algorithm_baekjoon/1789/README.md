# 백준 1789번 - 수들의 합

```
서로 다른 N개의 자연수의 합이 S.
S를 알 때, 자연수 N의 최댓값을 구하라

입력: 자연수 S가 주어진다
출력: 첫째 줄에 자연수 N의 최댓값 출력

```   


### 풀이 및 접근   
서로 다른 N개의 자연수의 합 = 1~n까지의 합을 의미한다.   
수열 공식 = (n * (n+1)) / 2을 이용하여 계산을 수행한다.   


반복문 수행 시 마지막 S의 개수에서 1을 뺀다.
이유는 n이 반복문을 돌면서 한번 더 수행하기 때문.
<br>   

### 소스코드   
```python
s = int(input())
n = 1

while (n*(n+1)) / 2 <= s:
    n += 1
print(n-1)
```
